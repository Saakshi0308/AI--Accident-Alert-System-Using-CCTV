# AI--Accident-Alert-System-Using-CCTV

## Problem Statement
Delay in accident reporting increases fatality rates.

## Solution
This project detects accidents automatically using AI and sends instant alerts with location details.

## Features
- Real-time accident detection
- Automatic alert system
- Location tracking
- AI-based classification

## Technology Used
- Python
- Machine Learning
- Sensors / GPS (if applicable)

## Future Scope
- Mobile application
- Integration with emergency services
